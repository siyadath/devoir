a)la commande git pull récupere les dernieres modifications depuis le depot distant  et les fusionne automatiquement avec la branche locale en cours.La commande git fetch recupere egalement les dernieres modifications mais ne les fusionne pas avec la branche locale.Elle met simplemeny a jour les references locales pour refleter les modifications du depot distant
b)Pour annuler les modifications locales non validees dans git vous pouvez l'utiliser avec la commande "git reset --hard".Cette commande annule les modifications non validées et restaure les fichiers a l'état ou ils se trouvaient lors du dernier commit.attention toutes les modifications non validées seront perdues avec cette commande donc assurez-vous que vous aviez bien sauvegardé tout ce que vous voulez conserver
c)Pour supprimer une branche distante dans Git vous pouvez utiliser la commande "git push origin nom_de_branche .Cette commande supprime la branche spécifiée sur le depot distant
d)Pour voir l'historique des modifications pour un fichier spécifique dans Git vous pouvez utiliser la commande "git log" nom_de_fichier cette commande affiche l'historique des modifications pour le fichier
